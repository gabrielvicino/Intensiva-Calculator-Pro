# Intensiva Calculator Pro ⚕️

Ferramenta para Médicos Intensivistas: Cálculos de Infusão Contínua, Intubação (IOT) e Conversões.

### 📋 Funcionalidades
* **Infusão Contínua:** Cálculo de vazão, concentração e doses de segurança.
* **Intubação (IOT):** Tabela de volumes baseada no peso do paciente.
* **Conversão Universal:** Transformação rápida entre unidades (mcg/kg/min, ml/h, etc).
* **Nuvem:** Banco de dados integrado e editável via Google Sheets.

### 🛠️ Tecnologias
* Python & Streamlit
* Google Sheets API (gspread)
* Pandas Dataframes
